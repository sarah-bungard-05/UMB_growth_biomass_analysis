# UMB_growth_biomass_analysis
Recreates figures and results from "Hydraulic Trait Strategies Drive Divergent Tree Growth Responses to Multidecadal Climate Variability " using US-UMB AmeriFlux data, diameter growth and biomass increment dataset from associated forest plot, and drought indices.

Requires data downloaded from Ameriflux Network (AmeriFlux BASE, US-UMB, https://ameriflux.lbl.gov/data/download-data/)
